IMPORTANT NOTE: !!!!!! READ THIS !!!!!
If after modding, the images look distorted or have wrong colors, then it is because your input image has more colors than the original image. Reduce the color count of your input image using GIMP or some online tool like https://onlinepngtools.com/decrease-png-color-count

RULES
1. Digimon Power needs to be between 0-225. Anything outside this range WILL BREAK YOUR Digivice.
2. While changing Digimon Names, your New Digimon Name should have the same length as your Old Digimon Name. Use _ for whitespace. Only letters allowed, no numbers or symbols.
3. While changing Digimon Images, your New Digimon Images should have the same color count as your Old Digimon Images. Use some tool like https://townsean.github.io/canvas-pixel-color-counter/ to find the color count of your images.
4. Take backups of bin
5. Do NOT abort any operation after it starts. Wait for it to complete.
